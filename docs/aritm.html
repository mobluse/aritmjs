Aritm - huvudräkningsträningsprogrammet

Finns för:
  1. Webbläsare för mobiler, surfplattor och datorer (kan läggas till på hemskärmen)
    1. Aritm Local (Använder Local Storage för att spara data mellan körningar och kan köras i flygläge.)
    2. Aritm Cloud (Lagrar data i Googles moln och man kan byta enhet och fortsätta med samma uppgifter.)
    3. Aritm SR Local (Med taligenkänning. Använder Local Storage för att spara data mellan körningar och kan köras i flygläge.)
    4. Aritm SR Cloud (Med taligenkänning. Lagrar data i Googles moln och man kan byta enhet och fortsätta med samma uppgifter.)
    5. Öppen källkod (GPLv3+) för Aritm Local och Aritm Cloud i mappen docs.
  2. Miniräknare och webbsidor med Python
    1. NumWorks
    2. trinket.io
  3. Miniräknare (ä.k. grafräknare) av märket CASIO: fx-7400GII, fx-9860GII, och troligen fx-9750GII
    1. M.O.B.
    2. casiocalc.org
  4. Miniräknare (ä.k. grafräknare) av märket Texas Instruments: TI-82, TI-82 STATS, TI-83, TI-83 Plus, TI-84 Plus (inklusive Silver Edition), och snart TI-89
    1. M.O.B.
    2. ticalc.org
  5. Datorer med stöd för BASICODE-3C och datorn Apple II
    1. BASICODE i webbläsare; även för iPhone- och Android-mobiler, ty skärmtangentbord (klicka Keyboard)
    2. Applesoft BASIC i webbläsare; även för iPhone-mobil, ty det inbyggda tangentbordet poppar upp
    3. Samma som ovan, men på GitHub
  6. Hemdatorerna VIC 20, Commodore 64 och 128 (VIC20, C64, & C128), samt emulatorer som THEC64 MINI, VICE, och Commander X16
    1. Commander X16 online emulator i webbläsare
    2. GitHub
    3. M.O.B.
    4. YouTube
  7. MMBasic för PicoMite och Linux (MMB4L)
    1. GitHub
  8. Datorer med TinyBASIC Two som bl.a. finns för Linux: t.ex. Raspberry Pi med Rasbian, och då även i konsolläge (d.v.s. fullskärm & utan X)
    1. M.O.B.
    2. YouTube
  9. GW-BASIC för MS-DOS (fungerar i DOSBox på många plattformar)
    1. GitHub
    2. M.O.B.
  10. Mobiltelefoner med Java ME (MIDlet)
    1. GetJar
    2. YouTube
    3. M.O.B. Demo i Java SE inkl. Android-version
  11. Webbläsare med Java SE
    1. M.O.B.
    2. YouTube
  12. Kalkylprogrammet LibreOffice.org/OpenOffice.org Calc som Basic-makron i ods-filer
    1. OOo.Orbin.SE
  13. MS-DOS (från 1992; fungerar i DOSBox)
    1. M.O.B.
    2. MS-DOS.Orbin.SE
